概述：
    留言板websocket api
请求url：
    ws://localhost:8000/ws/message/?report_id=
请求参数:
    reportId，置于如上url的report_id=
输入示例:
    ws://localhost:8000/ws/message/?report_id=100
    {
        'message': 'Hello World',
        'date': '2025-4-28 13:50' # 时间格式%Y-%m-%d %H:%M
    }
广播响应示例：
    {
        'message': {
            'username': '张三',
            'message': 'Hello World'
        }
    }
