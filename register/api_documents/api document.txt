概述：
    注册接口
请求url：
    https://localhost/api/users/register
请求方式：
    POST
参数：
    name(必须), password(必须且为明文密码（后端进行加密）)
输入示例:
    {
        'name': '张三',
        'password': '12345678'
    }
响应示例：
    成功验证：
    {
        'message': 'Success'
    }
    用户已存在：
    {
        'message': 'Existed'
    }
