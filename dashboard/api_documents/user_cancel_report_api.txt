概述：
    用户撤单api
请求url：
    https://localhost/api/dashboard/cancel_report/
请求方式：
    GET
参数：
    reportId
请求示例：
    {
        'reportId': 'reportId'
    }
响应示例：
    sessionid成功验证：
    {
        'message': 'Success'
    }
    sessionid过期：
    {
        'message': 'Session has expired'
    }
    sessionid不存在：
    {
        'message': 'Invalid session'
    }
    无cookies:
    {
        'message': 'No sessionid cookie'
    }
