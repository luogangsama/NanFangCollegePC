概述：
    修改值班时间端口
请求url：
    https://localhost/api/dashboard/change_duty_time/
请求方式：
    POST
参数：
    duty_time(值班日期，格式: int 1 - 5)
输入示例:
    {
        'duty_time': 5,
    }
响应示例：
    HTTP/1.1 200 OK
    Content-Type: application/json
    {
        'message': 'Success'
    }


错误响应：
    sessionid过期：
    HTTP/1.1 401 Unauthorized
    Content-Type: application/json
    {
        'message': 'Session has expired'
    }

    sessionid不存在：
    HTTP/1.1 401 Unauthorized
    Content-Type: application/json
    {
        'message': 'Invalid session'
    }

    无cookies:
    HTTP/1.1 400 Bad Request
    Content-Type: application/json
    {
        'message': 'No sessionid cookie'
    }

    身份错误:
    HTTP/1.1 403
    Content-Type: application/json
    {
        'message': 'Permission error'
    }
